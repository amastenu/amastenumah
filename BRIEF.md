# amastenumah.com Developer Brief

The full developer brief (Word document) has been shared with you separately.
This repo has been pre-structured per that brief.

## Key Rules
- ALL content/copy lives in the /content/ folder — never hardcoded in components
- Tailwind CSS only — no CSS modules, no styled-components
- No CMS
- No YouTube embeds for the hero video
- The EDITING-GUIDE.md must be completed before handoff

## Folder Structure
/content/     ← All editable text lives here
/components/  ← Reusable UI pieces
/app/         ← Pages
/public/      ← Images, videos, static files
